# Contrat API (OpenAPI)

Le backend FastAPI sera développé pour servir le frontend Streamlit en mode `BACKEND_MODE=api`.

- Spécification OpenAPI: `openapi.yaml`
- Authentification: header `Authorization: Bearer <token>`

## Comment lire / valider

- Ouvrir `openapi.yaml` dans Swagger Editor (ou une extension VS Code OpenAPI)
- L'objectif est de stabiliser les endpoints + schémas **avant** d'implémenter l'API.
